# Stock Monitor Automation Task - User Guide

## Overview
This set of scripts automates the Stock Monitor system (main.py), running it automatically on weekdays from 9:00 AM to 3:00 PM.

## Quick Start

### 1. Configure Automation Tasks

1. **Right-click** `setup_tasks.bat`
2. Select **"Run as administrator"**
3. Wait for script to complete, two tasks will be created

### 2. Verify Tasks

Open "Task Scheduler" (search "Task Scheduler"), in the task list you will see:
- `StockMonitor_Start` - Weekdays at 9:00 AM (window visible)
- `StockMonitor_Stop` - Weekdays at 3:00 PM (silent)

## Manual Operations

### Manual Start
Double-click `start_stock_monitor.bat`, the program will:
- Start the Stock Monitor system
- Display output in the console simultaneously
- Save output to log file in logs/ directory

### Manual Stop
Double-click `stop_stock_monitor.bat`, the program will:
- Find and terminate running main.py processes
- Display operation results

## Log Files

All run logs are saved in the logs/ directory with filename format:
```
stock_monitor_YYYYMMDD_HHMMSS.log
```

## Managing Task Scheduler

### View Tasks
Run in PowerShell:
```powershell
Get-ScheduledTask -TaskName 'StockMonitor*'
```

### Delete Tasks
Run in PowerShell:
```powershell
Unregister-ScheduledTask -TaskName 'StockMonitor*' -Confirm:$false
```

Or re-run `setup_tasks.bat`, it will delete old tasks and create new ones.

## Task Scheduler Details

- **Start Task**: Runs automatically at 9:00 AM, shows window so you can view real-time output
- **Stop Task**: Runs automatically at 3:00 PM, silently closes program

## Important Notes

1. **Administrator Rights**: Configuring Task Scheduler requires administrator rights
2. **Python Environment**: Ensure Python is installed and configured in system PATH
3. **Working Directory**: Scripts have correct directory paths configured by default, edit path variables in scripts if needed
4. **Log Output**: When running, output is displayed in console and saved to log file simultaneously
5. **Window Display**: Auto-start shows window for easy real-time output viewing
