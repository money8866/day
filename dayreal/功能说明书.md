# 股票盘中预警系统 - 功能说明书

## 1. 系统概述

本系统是一套完整的股票盘中预警和仓位管理工具，基于通达信行情数据接口，实现了个股预警、板块监控、四因子仓位动态优化等功能。

**主要特点：**
- 无次数限制的实时行情数据获取
- 支持概念板块整体监控
- 四因子动态仓位管理
- 定时仓位分析
- 灵活的配置管理

## 3. 核心功能详解

### 3.1 个股5日均线预警

**功能说明：**
监控个股价格接近5日均线的情况，触发预警提示。

**配置：**
- 配置文件：[stocks.csv](file:///workspace/stocks.csv)
- 配置格式：股票代码列表
- 检查间隔：默认10秒（可在[config.yaml](file:///workspace/config.yaml)中修改）

**预警条件：**
股票价格与5日均线的偏离度 ≤ 设定阈值（默认2%）

### 3.2 概念板块整体监控

**功能说明：**
通过概念板块名称一次性获取整个板块所有股票，监控板块整体异动。

**配置：**
- 配置文件：[concepts.csv](file:///workspace/concepts.csv)
- 配置格式：概念板块名称列表
- 检查间隔：默认60秒（可在[config.yaml](file:///workspace/config.yaml)中修改）

**板块分析内容：**
- 板块平均涨幅
- 涨停股票数量
- 板块龙头识别
- 领涨股票排名

### 3.3 四因子仓位管理系统

**核心公式：**
```
总仓位上限 = 情绪系数 × 主线系数 × 指数系数 × 回撤系数 × 最大仓位
最终建议仓位 = min(盘后建议仓位, 四因子计算仓位上限)
```

#### 因子详解

| 因子 | 功能 | 取值范围 | 说明 |
|------|------|----------|------|
| **情绪系数** | 反映市场情绪强弱 | 0.3-1.2 | 弱势市场0.3，强势市场1.2 |
| **主线系数** | 反映板块主线强弱 | 0.5-1.2 | 无主线0.5，主线明确1.2 |
| **指数系数** | 反映大盘指数表现 | 0.3-1.2 | 大跌0.3，大涨1.2 |
| **回撤系数** | 反映资金回撤情况 | 0.3-1.0 | 超回撤0.3，无回撤1.0 |

**情绪评分标准：**
- ≥60分：强势市场（情绪系数1.2）
- 30-59分：偏强市场（情绪系数1.0）
- 0-29分：震荡市场（情绪系数0.8）
- -30到-1分：偏弱市场（情绪系数0.5）
- ≤-30分：弱势市场（情绪系数0.3）

### 3.4 定时仓位分析

**分析时间点：**
- 10:00 - 开盘后第一个关键时点
- 11:30 - 上午收盘前
- 13:30 - 下午开盘后
- 14:30 - 尾盘前

**分析内容：**
1. 市场情绪分析（涨跌比、涨跌停数量、平均涨幅）
2. 四因子分解展示
3. 仓位上限计算
4. 最终建议仓位输出
5. 操作建议

**容错机制：**
- 时间窗口±2分钟内触发
- 每个时间点只分析一次，防止重复

## 4. 配置说明

### 4.1 系统配置 ([config.yaml](file:///workspace/config.yaml))

```yaml
files:
  concepts_csv: concepts.csv      # 概念板块配置文件
  stocks_csv: stocks.csv          # 个股配置文件
  position_csv: position_config.csv  # 仓位配置文件
warning:
  ma5_threshold: 0.02             # 5日均线预警阈值（2%）
  stock_check_interval: 10        # 个股检查间隔（秒）
  concept_check_interval: 60      # 板块检查间隔（秒）
notification:
  enabled: false                  # 通知功能开关
  method: print                   # 通知方式
```

### 4.2 概念板块配置 ([concepts.csv](file:///workspace/concepts.csv))

```csv

```

### 4.3 个股配置 ([stocks.csv](file:///workspace/stocks.csv))

```csv

```

### 4.4 仓位配置 ([position_config.csv](file:///workspace/position_config.csv))

```csv
date,base_position,notes
2026-05-24,0.5,震荡市建议半仓
2026-05-23,0.4,偏谨慎
```

**说明：**
- `date`：日期，格式YYYY-MM-DD
- `base_position`：盘后建议基础仓位，0-1之间
- `notes`：备注说明

## 5. 使用流程

### 5.1 快速开始

1. **安装依赖：**
```bash
pip install -r requirements.txt
```

2. **配置股票池：**
   - 编辑 [concepts.csv](file:///workspace/concepts.csv) 添加概念板块
   - 编辑 [stocks.csv](file:///workspace/stocks.csv) 添加个股
   - 编辑 [position_config.csv](file:///workspace/position_config.csv) 设置基础仓位

3. **运行程序：**
```bash
python main.py
```

### 5.2 程序运行流程

1. **初始化阶段：**
   - 加载配置文件
   - 连接通达信服务器
   - 获取概念板块股票列表
   - 计算历史5日均线数据

2. **实时监控阶段：**
   - 个股5日均线预警检查（每10秒）
   - 概念板块异动检查（每60秒）
   - 定时四因子分析（指定时间点）

3. **四因子分析阶段（定时触发）：**
   - 获取实时行情
   - 分析市场情绪
   - 获取板块主线数据
   - 获取指数表现
   - 计算仓位上限
   - 输出建议

## 7. 模块详解

### 7.1 [tdx_data.py](file:///workspace/tdx_data.py) - 通达信数据模块

**主要类：** `TdxData`

**核心方法：**
- `connect()` - 连接通达信服务器
- `get_stock_quotes()` - 获取股票实时行情
- `get_concept_stocks()` - 获取概念板块股票
- `get_history_kline()` - 获取历史K线数据
- `get_index_quotes()` - 获取指数行情
- `calculate_index_change()` - 计算指数平均涨幅

### 7.2 [technical_analysis.py](file:///workspace/technical_analysis.py) - 技术分析模块

**核心方法：**
- `calculate_ma()` - 计算移动平均线
- `calculate_change_percent()` - 计算涨跌幅
- `analyze_concept_intraday()` - 板块盘中分析
- `check_concept_movement()` - 板块历史异动检查

### 7.3 [warning_system.py](file:///workspace/warning_system.py) - 预警系统模块

**主要类：** `WarningSystem`

**核心方法：**
- `check_stock()` - 检查个股预警条件
- `check_concept_breakout()` - 检查板块异动
- `notify()` - 发送预警通知

### 7.4 [position_optimizer.py](file:///workspace/position_optimizer.py) - 仓位管理模块

**主要类：**
- `FactorCalculator` - 四因子计算器
- `MarketSentiment` - 市场情绪分析
- `FourFactorPositionManager` - 四因子仓位管理器
- `TradingTimeHelper` - 交易时间助手

### 7.5 [config.py](file:///workspace/config.py) - 配置管理模块

**主要类：** `Config`

**核心方法：**
- `load_config()` - 加载配置
- `get()` - 获取配置项
- `load_concepts()` - 加载概念板块
- `load_stocks()` - 加载个股
- `load_base_position()` - 加载仓位配置

## 9. 注意事项

1. **交易时间：** 程序只在交易时间运行（9:30-11:30, 13:00-15:00）
2. **网络连接：** 需要能连接到通达信服务器
3. **数据准确性：** 建议结合其他数据源验证
4. **风险提示：** 本系统仅供参考，不构成投资建议
5. **仓位管理：** 请根据自身风险承受能力调整仓位
